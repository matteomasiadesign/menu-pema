Abilita Ordinamento

Per salvare l'ordine personalizzato dei piatti, è necessario aggiungere la colonna position su Supabase.
Copia ed esegui questa riga all'interno dell'**SQL Editor** di Supabase:

ALTER TABLE menu ADD COLUMN position INT DEFAULT 0;

Eliminare questo piatto?

L'azione rimuoverà permanentemente il piatto selezionato dal menu digitale di PEMA.
